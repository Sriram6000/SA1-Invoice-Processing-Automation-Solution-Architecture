# Data

This directory is intended to organize all datasets required for solution development and consumption. To maintain clarity and scalability, the following structure is recommended:

# Folder Details

**04-data/**

**├──raw/**
  - Contains original source data in its unaltered form.

**├──processed/**
  - Holds datasets that have been cleaned, normalized, or transformed to support automation and solution workflows.

>[!Note]
>Actual organizational data is confidential and restricted for internal use only. Public repositories should only include placeholder or sample data in the samples/ folder to demonstrate structure and integration.
This directory showcases the folder and file structure only—no real data is shared.
